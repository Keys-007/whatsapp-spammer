# WhatsApp-Spammer
A simple Python program to spam WhatsApp Chats

## Requirments:
* Chrome WebBrowser
* python
* pip
* selenium  

### Installation:

#### Download or Clone the Repository

#### Extract it

#### Go to the directory

#### Install selenium using the following command:

>>pip3 install selenium

#### Run the bot using the following command:

>>python3 spammer.py
